# Personal-Website
<img width="800" height="450" alt="personal-website-preview" src="https://github.com/user-attachments/assets/20ad2d7d-2ae1-448d-86fe-0ef6f4239837" />

My personal website, a single-page static site built with plain HTML, CSS & JS.

## Features

- Animated starry background with pixel-art comets and spaceships
- Work history and education with company logos
- Social links (email, LinkedIn, GitHub, X)
- Responsive layout for mobile and desktop

## Deployment

Hosted on [Cloudflare Workers](https://workers.cloudflare.com/) via the static assets configuration in `wrangler.jsonc`. Pushes to `main` deploy automatically.
